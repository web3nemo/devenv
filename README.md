# devenv

一键配置与更新本地开发环境。

## Windows开发环境

管理员权限下命令行窗口内运行[init.bat](/win/init.bat)

基本功能包括：
- [x] 欢迎词和座右铭
- [x] scoop软件包自动安装与更新
- [x] vscode及插件自动安装
- [x] npm包自动安装
- [ ] 服务开发：WSL, Docker, Kubernetes
- [x] 版本控制：Git
- [x] 文档工具：Markdown, DrawIO
- [ ] 数据工程：Jupyter Notebook
- [ ] 原生开发：Rust, C/C++
- [ ] 服务开发：.net
- [ ] 应用开发：Android/iOS, Windows/Mac/Linux, Web
